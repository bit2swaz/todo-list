## ğŸš€ Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository
```bash
git clone https://github.com/bit2swaz/todo-list.git
cd todo-list
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm start
```

4. For production build
```bash
npm run build
```
